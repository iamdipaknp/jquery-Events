The unbind() method & the off() method removes event handlers from selected elements.
This remove all or selected event handlers, or stop specified functions from running when the event occurs.
Also unbind event handlers using an event object.
This is used to unbind an event from within itself (like removing an event handler after the event has been triggered a certain number of times).
